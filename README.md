## Setup
1. Clone the repo  
   `git clone https://github.com/Murad-Hasil/ai-chatbot-demo.git`

2. Install dependencies  
   `npm install`

3. Add your API key  
   Create `.env.local` and add `GEMINI_API_KEY=your_api_key_here`

4. Start development server  
   `npm run dev`
